Métodos

Document

  • Variavel global do Javascript que representa a página inteira.
  • Possui funções que permitem manipular os elementos html da página.

Para atribuir a uma variável pode ser utilizada conforme o exemplo abaixo.

getElementById

  • Método dentro do objeto "document".
  • Retorna um elemento da página utilizando como paramentro seu atributo "id".

Após isso podemos utilizar a variavel "obj_menu" para manipular as propriedades do elemento com id: "menu".
